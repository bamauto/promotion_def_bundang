## 주의사항 및 권장사항

### SEO 윤리 (White-Hat SEO)
✅ **추천**:
- 정확하고 유용한 정보 작성
- 실제 방문 경험 기반 리뷰
- 자연스러운 내부 링크
- 적절한 키워드 밀도 (1-2%)
- 정기적 업데이트 및 개선

❌ **피해야 할 것**:
- 가짜 리뷰 작성
- 키워드 스터핑
- 클로킹 또는 숨겨진 텍스트
- 과도한 외부 링크 교환
- 불법 백링크 구매

### 지역 비즈니스와의 관계
- 가게와 협력 관계 구축
- 상호 이익이 되는 방식 추구
- 리뷰의 정직성 유지
- 가게 권리 존중 (사진/정보 사용 허가)

### 시간 투자
- **초기 구축**: 100-150시간 (8주)
- **월간 유지보수**: 10-15시간
- **콘텐츠 작성**: 주 5-8시간
- **모니터링**: 주 2-3시간

## FAQ

### Q1: 모든 콘텐츠를 한번에 작성해야 하나?
**A**: 아니요. 필러 5개 → 상위 리뷰 10개 → 추가 가이드 순으로 단계적으로 진행하세요. 품질이 수량보다 중요합니다.

### Q2: 얼마나 빨리 순위가 올라올까?
**A**: 일반적으로 3개월 후 일부 장꼬리 키워드, 6개월 후 주요 키워드 TOP 10 진입이 현실적입니다.

### Q3: 특정 가게를 홍보할 수 있나?
**A**: 가능하지만 정직하게. 모든 리뷰는 사실 기반이어야 하고, 특정 가게만 과도하게 홍보하면 신뢰도가 떨어집니다.

### Q4: 네이버와 구글 중 뭐가 더 중요한가?
**A**: 한국에서는 네이버(60%)가 더 중요합니다. 하지만 둘 다 최적화해야 완벽합니다.

### Q5: 예산이 없으면 어떻게 하나?
**A**: 모든 기본 전략은 무료로 실행 가능합니다. 유료 도구는 3-6개월 후에 고려하세요.

## 1. 추천 가이드 페이지 템플릿 (Recommendation Guide Template)

### 제목 옵션
```
✅ 분당 [업종] BEST 10 추천 | 2024년 최신 정보
✅ 분당 [업종] 완벽 가이드 | 초보자부터 마니아까지
✅ 분당 [업종] 추천순위 | 가격·분위기·위치별 분석
✅ 분당 [업종] 선택 가이드 | [지역명] 모음
```

### 메타 설명 예시
```
"분당 [업종]의 TOP 10을 엄선했습니다.
위치, 가격, 분위기, 서비스 기준으로 추천하는
인기 가게 정보와 선택 가이드를 확인하세요."

(154자)
```

### 콘텐츠 아웃라인

```
H1: 분당 [업종] BEST 10 추천 | 2024년 최신 정보

소개 (150-200단어)
- 분당 [업종] 시장 현황
- 왜 분당 [업종]인가?
- 이 가이드의 활용법

H2: 분당 [업종] 지역별 현황

  H3: 중부 지역 ([지역명])
  H3: 동부 지역 ([지역명])
  H3: 북부 지역 ([지역명])

H2: 분당 [업종] 추천순위 TOP 10
  H3: 1위 [가게명] - [특징 1단어]
      - 위치: [상세주소]
      - 특징: [3줄]
      - 추천대상: [표시]
      - ⭐ 예상 평점: X.X/5

  H3: 2위 [가게명] - [특징 1단어]
      ... (반복)

H2: 분당 [업종] 선택 가이드
  H3: 분위기별 추천
      - "조용한 분위기": [가게 1-3개]
      - "활기찬 분위기": [가게 1-3개]
      - "프리미엄 분위기": [가게 1-3개]
      - "감성적 분위기": [가게 1-3개]

  H3: 가격대별 추천
      - "저가 (XX,000-XX,000원)": [가게]
      - "중가 (XX,000-XX,000원)": [가게]
      - "고가 (XX,000원 이상)": [가게]

  H3: 지역별 추천
      - "신세계백화점 근처": [가게]
      - "정자역 주변": [가게]
      - "미금역 주변": [가게]
      ... (지역별)

  H3: 시간대별 추천
      - "평일 저녁": [가게]
      - "주말 밤": [가게]
      - "늦은 시간": [가게]

H2: 분당 [업종] 방문 가이드
  H3: 예약 방법
      - 전화 예약: 장점/단점
      - 온라인 예약: 방법
      - 카톡 예약: 가능 가게
      - 현장 방문: 팁

  H3: 준비 사항
      - 신분증 필수 여부

  H3: 최고의 방문 시간
      - 최적 혼잡도
      - 최적 분위기
      - 최고의 서비스

  H3: 가격 관련

      - "다양한 옵션이?"

  H3: 서비스 관련
      - "음식/음료 종류?"
      - "특별한 서비스?"
      - "그룹 예약 가능?"

  H3: 실용 정보
      - "주차 가능?"
      - "신분증 필수?"

H2: 분당 [업종] 이용 팁
  H3: 시즌별 팁

      - "할인 정보 활용"

결론 (150-200단어)
- 분당 [업종]의 매력
- 추천 가이드 활용 안내
- 다음 단계 (예약/방문)

관련 링크

- [상세 리뷰 보기]
- [시즌별 가이드]
```

## 2. 상세 리뷰 페이지 템플릿 (Detailed Review Template)

### 제목 옵션
```
✅ [가게명] 하이퍼블릭 상세 리뷰 | 위치, 가격, 분위기
✅ 분당 [가게명] | [업종] 완벽 리뷰 및 후기
✅ [가게명] 방문기 | 분당 [업종] 솔직 리뷰
```

### 메타 설명
```
"[가게명] 하이퍼블릭 상세 리뷰.
위치, 교통, 가격, 시설, 분위기를
실제 방문 후기를 바탕으로 정리했습니다."

(120자)
```

### 콘텐츠 아웃라인

```
H1: [가게명] 하이퍼블릭 상세 리뷰 | 위치, 가격, 분위기

소개 (100-150단어)
- [가게명]이란?
- 왜 주목할 만한가?
- 누에게 추천하는가?

  H3: 위치 & 교통

      - 가장 가까운 역
      - 대중교통 이용법
      - 운전 방향
      - 소요 시간

  H3: 영업 정보
      - 영업시간
      - 휴무일
      - 주차 정보
      - 예약 가능 여부

  H3: 연락처
      - 전화번호
      - 카톡 ID
      - 인스타그램
      - 공식 웹사이트

H2: 시설 & 분위기 (500-700단어)
  H3: 첫 인상
      - 외부 모습
      - 입구 분위기
      - 첫 인상 점수

  H3: 실내 인테리어
      - 메인 홀
      - 테이블 배치
      - 인테리어 스타일
      - 조명
      - 프라이빗 공간 유무
      - 화장실

  H3: 음향 & 음악
      - 음악 장르
      - 볼륨 수준
      - DJ 퀄리티
      - 라이브 뮤직 유무

  H3: 분위기 평가
      - "조용함/활기" 정도
      - "프리미엄/캐주얼" 정도
      - "연령대 분석"
      - "고객층 분석"
      - 방문 시간별 분위기 변화

H2: 메뉴 & 가격대 (400-600단어)
  H3: 기본 이용료
      - 기본 료금
      - 최소 주문
      - 생맥주/와인/칵테일 가격

  H3: 시간대별 가격 (테이블)

      |--------|--------|------|---------|
      | 평일 저녁 | | | |
      | 금토 저녁 | | | |
      | 늦은 시간 | | | |

  H3: 음료 옵션
      - 주류 종류
      - 추천 칵테일
      - 논알콜 옵션
      - 프리미엄 술 여부

  H3: 음식 메뉴
      - 대표 안주
      - 추천 메뉴
      - 맛 평가

H2: 서비스 & 경험 (400-500단어)
  H3: 스태프 서비스

      - 친절도
      - 응답 시간
      - 서빙 퀄리티

  H3: 고객 경험
      - 대기 시간
      - 예약 처리
      - 특별 요청 수용도
      - 환경 청결도

  H3: 방문객 프로필
      - 연령대
      - 성별 비율
      - 직업군
      - 방문 목적 추정

H2: 가격대 분석 (300-400단어)
  H3: 비용 분석
      - "1인당 평균 비용"
      - "4명 그룹 기준"
      - "평가: 가성비 (높음/중간/낮음)"

  H3: 경쟁사 비교
      - [경쟁사 1] vs [가게명]
      - [경쟁사 2] vs [가게명]
      - [경쟁사 3] vs [가게명]

H2: 방문 팁 (300-400단어)
  H3: 최고의 방문 시간

      - 신분증 필수 여부
      - 복장 규정

  H3: 주차 & 접근
      - 주차장 위치
      - 주차료
      - 대중교통 추천
      - 주변 편의시설

H2: 종합 평가 (200-300단어)
  H3: 총평
      - 별점: ⭐ X.X/5
      - 추천도: X/10
      - 재방문 의사: Yes/No

  H3: 추천 대상
      - "이런 사람에게 추천"
      - "이런 목적에 최적"
      - "피해야 할 대상"

  H3: 장점 요약

  H3: 개선 바라는 점
      - 아쉬운 점 2-3가지

  H3: 유사 가게 비교
      - "비슷한 가게 3곳"
      - "가장 유사한 가게"

H2: 방문객 후기 (선택사항)
  - 별 5개 후기 인용 (1개)
  - 별 3개 후기 인용 (1개)
  - 별 1개 후기 인용 (1개)
```

## 3. 비교 페이지 템플릿 (Comparison Template)

### 제목 옵션
```
✅ 분당 하이퍼블릭 vs 호스트바 | 5가지 차이점 완벽 비교
✅ 분당 가라오케 vs 셔츠룸 | 어떤 것을 선택할까?
✅ [가게명1] vs [가게명2] | 상세 비교 분석
```

### 메타 설명
```
"분당 하이퍼블릭과 호스트바의 차이점을
5가지 기준으로 완벽 비교했습니다.
당신에게 맞는 선택을 도와드립니다."

(95자)
```

### 콘텐츠 아웃라인

```
H1: 분당 하이퍼블릭 vs 호스트바 | 5가지 차이점 완벽 비교

소개 (200-300단어)

  H3: 하이퍼블릭이란?

      - 특징
      - 문화적 배경

  H3: 핵심 차이점
      - [포인트 1]

H2: 비교표 (큰 테이블)
  | 항목 | 하이퍼블릭 | 호스트바 | 승자 |
  |-----|----------|----------|------|
  | 가격 | | | |
  | 분위기 | | | |
  | 서비스 | | | |
  | 대상층 | | | |
  | 추천도 | | | |

H2: 상세 비교 분석
  H3: 1. 가격대 비교
      - 하이퍼블릭 평균 가격
      - 호스트바 평균 가격
      - 어디가 저렴한가?
      - 가성비 분석

  H3: 2. 분위기 비교
      - 하이퍼블릭 분위기
      - 호스트바 분위기
      - 선호도 분석
      - 상황별 적합도

  H3: 3. 서비스 비교
      - 하이퍼블릭 서비스
      - 호스트바 서비스
      - 차이점 분석
      - 서비스 품질

  H3: 4. 고객층 비교
      - 하이퍼블릭 방문객
      - 호스트바 방문객
      - 연령대 차이
      - 목적 차이

  H3: 5. 방문 목적별 추천
      - "회식 목적": 어느 쪽?
      - "데이트": 어느 쪽?
      - "친구 모임": 어느 쪽?
      - "1인 방문": 어느 쪽?

H2: 상황별 선택 가이드
  H3: 이렇다면 하이퍼블릭

  H3: 이렇다면 호스트바

      - "특별한 경험을 원한다면"
      - "[상황 4]"

H2: 분당 지역별 추천
  H3: 하이퍼블릭 추천 TOP 3
      - [가게 1]
      - [가게 2]
      - [가게 3]

  H3: 호스트바 추천 TOP 3
      - [가게 1]

      - [가게 3]

H2: 결론 & 최종 조언

  - 둘 다 방문해보기
```
